# X_O
## Описание 
Простейшая компьютерная игра "крестики-нолики" для проведения  досуга в компании с друзьями, или со своим компьютером если вы волк-одиночка 
## Цель проекта
Разработка кроссплатформенной игры "крестики-нолики", благодаря библеотеке Tkinter 
## Архитектура проекта
├── src/   
│   └── x_0.py
├── Dockerfile
├── README.md
├── .gitignore
├── requirements.txt
## Локальный запуск
# Установка зависимостей
pip install -r requirements.txt
## Примеры использования
Ну вот короче вы сидите вечером, вам отключили интернет. В доту и кс не поиграть, а у вас на компе как раз есть свежайшая сочная версия игры крестиков-ноликов, и ваш унылый скучный вечер превращается в унылый скучный вечер но с крестиками-ноликами
